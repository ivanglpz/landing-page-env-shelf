# env-shelf landing

Landing page built with Astro + TypeScript + Tailwind CSS.

## Stack

- Astro
- TypeScript (strict config)
- Tailwind CSS

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Structure

- `src/layouts/MainLayout.astro`: Base document layout and metadata.
- `src/components/*`: Reusable landing sections.
- `src/pages/index.astro`: Page composition.
- `src/styles/global.css`: Global styles and theme tokens.

## Notes

- Theme is dark by default.
- Content is in English.
- Sections cover env-shelf capabilities, i18n support, and open-source positioning.
